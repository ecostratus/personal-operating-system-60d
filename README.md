# Personal Operating System - 60 Day Execution Plan

A comprehensive personal operating system designed to support a 60-day execution plan for career development, job search, and consulting opportunities.

## Overview

This repository contains a structured framework that is:
- **Auditable**: All actions and decisions are tracked and documented
- **Reversible**: Changes can be rolled back and reviewed
- **Manually Overridable**: Human oversight is maintained throughout
- **Compatible**: Works with Microsoft 365, Copilot Studio, and lightweight scripting

## Repository Structure

```
.
├── docs/                      # Core documentation and governance
├── automation/                # Python automation scripts
│   ├── job-discovery/
│   ├── resume-tailoring/
│   ├── outreach/
│   ├── consulting-funnel/
│   └── interview-prep/
├── copilot-flows/            # Copilot Studio flow definitions
├── prompts/                  # AI prompt templates
├── excel-templates/          # Excel-based system of record
├── config/                   # Configuration files
└── tests/                    # Test suite
```

## Getting Started

1. Review the documentation in `docs/master-brief.md`
2. Configure your environment using `config/env.sample.json`
3. Set up Excel templates from `excel-templates/`
4. Review and customize automation scripts in `automation/`

## Key Features

- **Job Discovery**: Automated job scraping and scoring
- **Resume Tailoring**: AI-assisted resume customization
- **Outreach Management**: Template-based outreach generation
- **Consulting Funnel**: Proposal and offer generation
- **Interview Prep**: Structured interview preparation
