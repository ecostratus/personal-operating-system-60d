# Architecture Principles

These principles are durable and should rarely change.

## P1. Evidence Precedes Inference

No inference is valid without source-linked evidence.

## P2. Claims Are the Primary Reasoning Construct

Claims are the atomic unit of platform intelligence. Scores are derived artifacts.

## P3. Recommendations Must Be Explainable

Every recommendation must include positive and negative drivers with supporting claim and evidence references.

## P4. Policies Determine Optimization Behavior

Optimization goals and trade-offs are policy-configured data, not hard-coded constants.

## P5. Domain Knowledge Is Separate from Core Reasoning

Core engine logic is generic. Domain-specific mappings and archetypes are data packs.

## P6. Extensible Core, Focused Implementation

The data model is extensible. v1 implementation is intentionally scoped.

## P7. Deterministic Decision Layer

Decision outputs are deterministic for identical inputs.
